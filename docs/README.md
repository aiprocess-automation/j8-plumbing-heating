# Documentation

Two audiences, two folders.

| Folder | For | Read this if you are… |
| --- | --- | --- |
| [`technical/`](./technical/) | AI agents, operators, developers | Implementing the upstream generator pipeline, modifying the template, debugging deployments, or trying to understand the architecture |
| [`buyer/`](./buyer/) | The local-business owner who bought a site | Logging into the admin for the first time, editing your site, connecting Google, or troubleshooting basic issues |

If you're a **buyer** and you found this folder by accident, the file you want is [`buyer/README.md`](./buyer/README.md). It's written in plain English, no jargon.

If you're an **operator or AI agent** working on the template itself or its generation pipeline, start with [`technical/architecture.md`](./technical/architecture.md).
